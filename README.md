# Autoprompter

A teleprompter that lives in your MacBook's notch. It listens to you speak and follows along — even if you go off-script.

## How it works

1. Hover over the notch to open the panel
2. Paste your script
3. Hit **Start**
4. The highlighted line follows your voice in real time

You can paraphrase, skip ahead, or ad-lib. It keeps up.

## Setup

You need a free [Deepgram API key](https://console.deepgram.com) for speech recognition.

```sh
swift run
```

The app will ask for your key on first launch.

## Requirements

- MacBook with macOS 15+
- Xcode 16+ (or just `swift run` from Terminal)
